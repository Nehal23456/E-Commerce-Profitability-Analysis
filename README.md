📊 E-Commerce Profitability Analysis

📌 Project Overview

This project analyzes the profitability of an e-commerce business using SQL, Python, and Power BI. The objective is to identify revenue drivers, evaluate product and sales channel performance, measure marketing effectiveness, and uncover actionable business insights that can improve profitability.

The project follows a complete data analytics workflow—from data preparation and SQL analysis to interactive dashboard creation in Power BI.


 🎯 Objectives

* Analyze overall sales performance.
* Calculate revenue, costs, and profit across products.
* Identify the most and least profitable product categories.
* Compare profitability across different sales channels.
* Evaluate marketing campaign effectiveness.
* Generate business insights for strategic decision-making.

 🛠️ Tech Stack

* **SQL** – Data querying and business analysis
* **Python (Pandas)** – Data loading and preprocessing
* **Power BI** – Interactive dashboard creation
* **Microsoft Excel** – Source datasets
